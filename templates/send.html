Mbtify



MBTI Classification Using Track Link : Only for Spotify


Mbtify를 이용하시려면 먼저 스포티파이 계정이 있어야 합니다.


이미지가 보이지 않는다면, 이 링크를 참고 하거나 아래 Step을 밟아보세요.


    1. 스포티파이 트랙에서 오른쪽 마우스를 클릭하면 옵션이 나온다.

    2. 공유하기 탭을 누릅니다.

    3. 곡 링크 복사하기를 누릅니다.

    4. 아래 입력 란에 복사한 링크를 넣고 예측을 시작합니다.

곡 링크 입력 :



상위 그림과 같이 Spotify Track ID를 올바르게 입력해주시면 됩니다.
